# AI Powered Intelligent Decision Making System - Thesis Prototype

This repository contains the AI-powered CRM Assistant developed for the thesis research on reducing Turnaround Time (TAT) in financial institutions.

## 🚀 How to Run the App
1. Make sure you have Python installed.
2. Install the required libraries:
   ```bash
   pip install streamlit pandas
   ```
3. Run the application:
   ```bash
   streamlit run CRM_AI_Assistant_App.py
   ```

## 🛠️ Key Features
- **💬 AI Mentor Tab:** Natural language query system for Home, Auto, and SME manuals.
- **💰 Eligibility Calculator:** Real-time BTR & Max Loan calculator with 45% DBR logic.
- **🌐 Compliance Watcher:** Live toggle for Bangladesh Bank circular alignment.
- **📜 Policy Timeline:** Historical rule explorer (2022 - 2026).
- **📥 Document Center:** Links to official central bank publications.
